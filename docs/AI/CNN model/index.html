CNNモデルってなに？

🧠 ニューラルネットワークって？​

まず、ニューラルネットワークは、人間の脳の神経細胞（ニューロン）のつながりをまねた機械学習のモデルです。画像や音声などのデータから特徴を学んで、分類や認識をします。

🖼️ CNN（畳み込みニューラルネットワーク）とは？​

CNNは画像認識に強いニューラルネットワークの一種です。

  1. 畳み込み（Convolution）
    🧩 小さな「フィルター（カーネル）」を画像の上に乗せて、特徴を抽出します。
    例えば、エッジや色の変化、形のパターンを見つける役割です。

  2. 活性化関数（Activation）
    ⚡ 抽出した特徴をわかりやすく変換します。
    代表的なものに「ReLU（レル）」があります。
    「0より小さい値は0にするよ」という働きで、非線形の複雑なパターンを学べます。

  3. プーリング（Pooling）
    🔍 画像の特徴マップを縮小して重要な情報だけ残します。
    代表例は「最大プーリング」で、周りの中で一番大きい値だけを残します。

  4. 全結合層（Fully Connected Layer）
    🎯 最後に抽出された特徴をまとめて、「これは何の画像か？」を判定します。

例）猫と犬を見分ける場合​

まず、耳の形や目の位置などの特徴をフィルターで抽出。

いくつかの層で情報をまとめていき、 最終的に「これは猫です」「これは犬です」と判断します。

CNNの特徴​

ポイント説明
🧩 畳み込み小さな窓で特徴を見つける
⚡活性化関数特徴をわかりやすく変換
🔍プーリング重要な情報だけを残す
🎯全結合層最終判断をする

代表的なCNNモデルまとめ​

モデル名発表年特徴・ポイント用途例
ResNet2015年深い層でも学習しやすい「残差接続（Residual Connection）」を導入。非常に高い精度を実現。画像分類、物体検出、医療画像解析
MobileNet2017年軽量かつ高速な畳み込みを実現する「深さ方向別畳み込み（Depthwise Separable Convolution）」を採用。スマホアプリ、組み込み機器、リアルタイム処理

ResNetとは？​

  • 残差接続（Residual Connection） *によって、深い層でも勾配消失問題を軽減し、効率的な学習を可能に。
  • ボトルネック構造（ResNet-50以降）で計算コストを抑えつつ深いネットワークを実現。
  • 転移学習のベースモデルとして幅広く利用されている。

*残差接続：深いニューラルネットワークの学習をうまく進めるための仕組み。
深いニューラルネットワークでは、情報や勾配が層を通るうちに失われやすく、層を増やすと性能が悪化することがあります。
「何かすごい処理をやってみたけど、もしかしてそのままの方がよかったかも……」
→ これに備えて元の情報を残しておく！つまり、「やりすぎない」工夫。

📊 各バージョンの比較​

バージョン発表年主な特徴・ポイント用途例
ResNet-182015年18層の比較的浅いネットワーク。軽量で実装・学習が容易。教育用途、軽量タスク、基礎実験
ResNet-342015年34層。ResNetの基本構造を維持しつつ、より深い層で高精度を実現。中規模の画像分類タスク
ResNet-502015年ボトルネック構造（1x1畳み込みを使った層削減）を導入し、計算効率を改善。画像分類、物体検出、転移学習
ResNet-1012015年101層の深いネットワーク。より複雑な特徴を抽出可能。高精度が求められる画像認識
ResNet-1522015年152層。最も深い標準版ResNet。精度を最大化したい場合に使用。医療画像解析、大規模画像分類

MobileNetとは？​

MobileNetシリーズは、軽量で高速なモデル設計を目的としたCNNファミリーです。
バージョンごとに計算効率や精度が進化しています。

📊 各バージョンの比較​

比較ポイントMobileNetV1MobileNetV2MobileNetV3
計算効率良い良い＋改良さらに最適化
精度基本的V1より高いV2よりさらに高い
モデルサイズ小さいほぼ同等または少し大きい同等かやや軽量
適した用途軽量処理が必要な組み込み機器高速かつ精度が欲しいスマホや端末省電力で高精度なスマホやIoT端末

📝 MobileNetまとめ​

  • MobileNetV1 は「軽量化の基本形」
    → モデル圧縮の基礎。シンプル構造で高速。

  • MobileNetV2 は「高性能化＋軽量維持」
    → 残差構造を工夫し、V1よりも精度アップ。

  • MobileNetV3 は「AIが設計して性能と効率を最適化」
    → NAS（Neural Architecture Search）で自動設計され、性能・効率・省電力を両立。
